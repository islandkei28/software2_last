# software2_last

宿題リポジトリの期限が2026/01/29 14:59 UTCで期限切れと表示されてしまったので、自分のリポジトリで作成しました。ご容赦ください。

# 実行方法
gcc -Wall -o mylifegame mylife.c<br>
./mylifegame

# a.ランダム配置
time.hをincludeしsrand(time(NULL))をmain内に置いた。
20%の確率で画面上に発生するようにした。

# b.ファイル出力
my_save_cells_lif(・)を追加

# c.独自の工夫
過密状態で爆発して消滅させるようにした。過密の定義（周囲がnマス以上に存在していること）は変数OVERCROWDから容易に変更が可能になっている。初期設定は6になっている。

# 展望
地形等の環境依存性は面白そう。
平地はa.ランダム配置の時点で生まれる確率が高くなったり、近くに資源（水や食料）があれば発生する条件や消滅する条件が緩くなったりする設定ができる。
